Bootstrap Double Sidebar


This template has responsive menus toggling system on both sides. Both the menus will appear collapsed on smaller screens, and will appear non-collapsed on larger screens.

The menus can be toggled manually with the and buttons positioned at bottom corners of each sides. When toggled using the buttons, the menu will appear/disappear. On small screens, the page content will be pushed off canvas.

The menu can have as many sub-menu as possible, as shown in this demo page. This is achieved using Bootstrap's Collapse.

This project is forked from Simple Sidebar of Start Bootstrap.
